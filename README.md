# Learning French (A2 Level)

This repository contains a curated set of resources for learning French, specifically tailored for moving from an A1 foundation to A2 proficiency.

## Repository Structure

- **[vocabulary/](file:///Users/mp/Documents/LearningFrench/vocabulary/)**: Foundational and intermediate word lists with example sentences.
  - [a1.md](file:///Users/mp/Documents/LearningFrench/vocabulary/a1.md): Essential basics.
  - [a2.md](file:///Users/mp/Documents/LearningFrench/vocabulary/a2.md): Work, travel, social life, and feelings.
- **[grammar/](file:///Users/mp/Documents/LearningFrench/grammar/)**: Key structural concepts.
  - [sentence_structures.md](file:///Users/mp/Documents/LearningFrench/grammar/sentence_structures.md): Passé Composé vs. Imparfait, comparisons, and relative pronouns.
  - [verb_conjugations.md](file:///Users/mp/Documents/LearningFrench/grammar/verb_conjugations.md): Essential verb charts for multiple tenses.
- **[conversation/](file:///Users/mp/Documents/LearningFrench/conversation/)**: Practical speaking tools.
  - [daily_phrases.md](file:///Users/mp/Documents/LearningFrench/conversation/daily_phrases.md): Common reactions and filler words.
  - [examples.md](file:///Users/mp/Documents/LearningFrench/conversation/examples.md): Real-world dialogues (ordering food, asking directions, etc.).

## How to Get Started

1. Check your foundation in the `vocabulary/a1.md` file.
2. Study the `grammar/sentence_structures.md` to understand how to talk about the past.
3. Use the `conversation/examples.md` to practice real-life scenarios.

Bonne chance !
